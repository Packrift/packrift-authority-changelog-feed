# Packrift Authority Changelog Feed

This repository is the public source-of-record snapshot for the Packrift Authority Changelog Feed static corpus.

Canonical public feed:

- https://packrift-authority-changelog-feed.vercel.app/

The corpus publishes sanitized Packrift authority-ledger movements as public web pages and machine-readable feed files. It intentionally includes only the public static payload:

- HTML routes for the feed, update list, verification policy, feed formats, and count boundaries
- `llms.txt` and `llms-full.txt`
- RSS, Atom, JSON Feed, and ActivityStream outputs
- public data files under `data/`
- sitemap, robots, OpenAPI, and agent-manifest metadata

Build scripts, deployment logs, local QA evidence, verification artifacts, private workspace paths, and non-public commerce details are not part of this repository.

Count treatment:

- This GitHub repository is one third-party source surface after public readback.
- Supporting raw files, GitHub API responses, and duplicate raw URLs are evidence only and are not separate Packrift authority-count rows.
- Any Software Heritage archive of this origin is counted separately only after a completed public archive readback.
